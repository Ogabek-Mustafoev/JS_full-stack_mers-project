<h2>🛠️ Installation Steps:</h2>

<p>1. Download code</p>

```
git clone https://github.com/Ogabek-Mustafoev/JS_full-stack_mers-project.git
```

<p>2. Install packages</p>

```
npm install
```

<p>3. Run Json-server:</p>

```
npx json-server db.json
```
